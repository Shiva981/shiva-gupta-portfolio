A modern, responsive personal portfolio built with React, TypeScript, and Vite.

This portfolio showcases my work and interests in:

- Electrical Engineering
- Robotics
- Embedded Systems
- PCB Design
- Software Development
- AI Automation

## ✨ Features

- Responsive design
- Dark theme
- Smooth animations
- Interactive project showcase
- Skills & technologies section
- Contact information
- Scroll progress indicator
- Back to top button
- Custom cursor
- Modern UI

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- CSS3
- Framer Motion
- React Icons

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Shiva981/shiva-gupta-portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
